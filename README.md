## Contributing

Thank you for considering contributing to the Lumynus framework! The contribution guide can be found in the [Lumynus documentation](https://lumynus.dev/docs/contributions).

## Code of Conduct

In order to ensure that the Lumynus community is welcoming to all, please review and abide by the [Code of Conduct](https://lumynus.dev/docs/contributions#code-of-conduct).

## Security Vulnerabilities

If you discover a security vulnerability within Lumynus, please send an e-mail to the maintainers via [security@lumynus.dev](mailto:security@lumynus.dev). All security vulnerabilities will be promptly addressed.
